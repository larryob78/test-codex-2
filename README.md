# Social Studio AI

A lightweight, browser-based creative studio that drafts copy and generates ready-to-post visuals for every major social format.

## Getting started

Open `index.html` in your browser or run a simple static server:

```bash
python3 -m http.server 8000
```

Then visit [http://localhost:8000](http://localhost:8000) to launch the app.

## How it works

1. Enter your brand name, a campaign idea, choose a tone, and optionally a call to action.
2. Social Studio AI instantly crafts platform-specific headlines, captions, CTAs, and hashtags.
3. Each placement also renders an AI-inspired gradient image tailored to the brand palette.

Export the copy or download the visuals to drop straight into your social scheduler.
