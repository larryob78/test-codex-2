# Social Studio AI

A lightweight, browser-based creative studio that drafts copy and generates ready-to-post visuals for every major social format.

## Live preview

Spin up a live demo with a shareable tunnel URL:

```bash
npm install
npm run preview:tunnel
```

The command serves the app at [http://localhost:4173](http://localhost:4173) and prints a public forwarding link you can share to preview the experience remotely. Hit `Ctrl+C` when you're done to close both the server and tunnel processes.

## Local development

If you only need a local session, run:

```bash
npm install
npm run preview
```

Then visit [http://localhost:4173](http://localhost:4173) in your browser to launch the app.

## How it works

1. Enter your brand name, a campaign idea, choose a tone, and optionally a call to action.
2. Social Studio AI instantly crafts platform-specific headlines, captions, CTAs, and hashtags.
3. Each placement also renders an AI-inspired gradient image tailored to the brand palette.

Export the copy or download the visuals to drop straight into your social scheduler.
