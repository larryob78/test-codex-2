# API Overview

Base URL: `http://localhost:4000`

## Endpoints
- `POST /v1/campaigns` create campaigns
- `GET /v1/campaigns` list campaigns
- `POST /v1/ai/generate-copy` AI copy assistance
- `POST /v1/ai/generate-image` AI image assistance
- `POST /v1/ai/generate-video` AI video assistance
- `POST /v1/privacy/export` DSAR export
- `POST /v1/privacy/erase` DSAR erasure
- `GET /healthz` health status
